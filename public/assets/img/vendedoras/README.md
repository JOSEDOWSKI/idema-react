# Fotos de Vendedoras

Este directorio debe contener las fotos de las vendedoras del sistema de WhatsApp.

## Archivos actuales:
- `dayana.jpeg` - Foto de Dayana (30KB)
- `alessandra.jpeg` - Foto de Alessandra (34KB)
- `malu.jpeg` - Foto de Malu (123KB)
- `rosh.PNG` - Foto de Rosh (881KB)

## Archivos requeridos (formatos aceptados):
- `dayana.jpg`, `dayana.jpeg`, `dayana.png` o `dayana.PNG` - Para Dayana
- `alessandra.jpg`, `alessandra.jpeg`, `alessandra.png` o `alessandra.PNG` - Para Alessandra
- `malu.jpg`, `malu.jpeg`, `malu.png` o `malu.PNG` - Para Malu
- `rosh.jpg`, `rosh.jpeg`, `rosh.png` o `rosh.PNG` - Para Rosh

## Especificaciones:
- Formato: JPG, JPEG, PNG (mayúsculas o minúsculas)
- Tamaño recomendado: 200x200 píxeles
- Peso máximo: 100KB por imagen (recomendado)
- Aspecto: Cuadrado (1:1) recomendado

## Nota:
Si las fotos no están disponibles, el sistema mostrará iconos por defecto.
